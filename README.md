## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
* nodejs 12.x
* npm - to install packages locally

### Setup
* git clone https://github.com/AxelSoler/space-travelers-hub.git

### Install
* npm install

### Lint code
* npm run eslint
* npm run stylelint
